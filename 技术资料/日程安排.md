  ```
  广度优先搜索：用一个queue来实现 
  public void levelOrderTraversal(ListNode node){
      if(node==null){
      	System.out.println("empty,tree");
      	return;
      	}
      	Queue<Node> queue=new LinkedList<Node>();
      	queue.add(node);
      	while(!queue.isEmpty()){
              ListNode temp=queue.poll();
              if(temp.left!=null)
              queue.add(temp.left);
              if(temp.right!=null)
              queue.add(temp.right);
              
      	}
      	
  }
  ```
